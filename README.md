# Acme Store

A modern web application powered by 
- [Vite](https://vitejs.dev/)
- [React](https://react.dev/)

## Getting Started

1. **Install dependencies:**
  ```bash
  npm install
  ```

2. **Run the development server:**
  ```bash
  npm run dev
  ```

3. **Build for production:**
  ```bash
  npm run build
  ```

## Features

- View products, details
- Add to cart
- Purchase
